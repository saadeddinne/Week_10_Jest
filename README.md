## JS AND Jest

- This module use 2 APIs as media source Open Weather and Bitcoin API
- The APIs are used by axios and the test are made by Jest .

## For Testing

- Do an npm install for installing all the project dependencies:

```sh
npm install
```

- To run the app:

```sh
node streamer.js
```

- To TEST the app:

```sh
npm test
```
